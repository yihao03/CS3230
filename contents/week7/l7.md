1. Cast the problem where we have to make a choice and are left with
one subproblem to solve.
2. Prove that there is always an optimal solution to the original
problem that makes the greedy choice, so the greedy choice is safe.
3. Use optimal substructure to show that we can combine an optimal
solution to the subproblem with the greedy choice to get an
optimal solution to the original problem.

# Dynamic vs Greedy

- Dynamic: avoid overlapping work in optimal substructure
- Greedy: go down on the optimal path on each step i.e. no 'useless work'
